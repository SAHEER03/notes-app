# notes-app
Full Mern-Stack Project (React, Express, Node.js, MongoDB)
Users can createnotes, search notes, delete notes and colloborate with other users by adding them as colloborators


# Features
JWT used for User Authentication
Complete Register and Login system
Create notes with a rich text editor
Delete notes
Full-text search 
Add collaborators to note
Notes visible only to owner and collaborators
UI built with TailwindCSS

# Tech stacks
    # Frontend
        React
        Axios
        TailwindCSS
    # Backend
        Express.js
        Node.js
        MongoDB
        Mongoose
    # Authentication
        JSON web token(JWT)

# Assumptions
Collaborators are added using only email
Notes can be accessed by owner and collaborators
JWT tokens are stored in browser
